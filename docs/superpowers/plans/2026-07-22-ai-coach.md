# TypePilot AI Coach (Gemini) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add AI coaching to the finished-test experience — a short post-test analysis card (strength/weakness/recommendation) and a personalized practice passage generated from the user's weak spots, fed into the existing typing engine — powered by Google Gemini via a small backend that holds the API key.

**Architecture:** A new Node + Express API server (`server/`) holds `GEMINI_API_KEY` and calls Gemini through the `@google/genai` SDK, exposing `POST /api/analyze` (structured JSON) and `POST /api/passage` (text). All Gemini SDK specifics live in one adapter file (`server/gemini.ts`); routes, prompts, schema, validation, and the client are provider-agnostic and unit-tested with the adapter/fetch mocked. The Vite dev server proxies `/api/*` to the API server; both run under one `npm run dev`.

**Tech Stack:** Existing Vite + React + TS + Vitest, plus `express`, `cors`, `@google/genai`, `tsx`, `concurrently`, `dotenv`, `supertest`.

## Global Constraints

- **Never send keystrokes or raw typed text to the AI** — only the compact `CoachInput` summary (WPM, accuracy, consistency, weak letters, slow words, backspace count, recent average).
- **No AI calls during a test** — only after completion, from the results screen.
- **AI is additive** — the results screen and history must render fully even when the AI call fails, times out, or the key is missing. Show a friendly fallback, never an error crash.
- **The API key never reaches the browser** — it lives only in the server's environment.
- **No live LLM calls in tests** — mock `server/gemini.ts` (server) and `fetch` (client). Everything deterministic.
- All Gemini SDK usage is confined to `server/gemini.ts`. No other file imports `@google/genai`.
- Model is env-configurable via `COACH_MODEL` (default `gemini-3.6-flash`).
- TypeScript strict. `src/engine/` and `src/storage/` stay React-free and unchanged except where a task explicitly says so.
- Commit after each task.

```ts
export type CoachInputShape = {
  netWpm: number
  grossWpm: number
  accuracy: number
  consistency: number
  weakLetters: string[]
  slowestWords: string[]
  backspaceCount: number
  recentAverageWpm: number
  mode: 'words' | 'quotes'
  durationSeconds: 30 | 60
}

export const ANALYSIS_SYSTEM = [
  'You are a supportive typing coach.',
  'Analyse the user\'s typing performance using the provided statistics.',
  'Return: one strength, one primary weakness, a concise explanation folded into',
  'the recommendation, one specific action the user should take next, and a',
  'recommended exercise type (a short snake_case tag such as "punctuation_accuracy"',
  'or "weak_letter_drill").',
  'Avoid generic advice. Do not shame the user. Keep the whole response under 100 words.',
].join(' ')

export const PASSAGE_SYSTEM = [
  'You generate natural typing-practice passages.',
  'Repeat the focus patterns naturally, avoid meaningless word combinations,',
  'match the user\'s difficulty level, and do not exceed the target length by more',
  'than 10%. Return only the passage text — no preamble, no quotes, no explanation.',
].join(' ')

export function buildAnalysisInput(input: CoachInputShape): string {
  return `Typing data:\n${JSON.stringify(input, null, 2)}`
}

export function buildPassageInput(input: CoachInputShape): string {
  const letters = input.weakLetters.join(', ') || 'none'
  const words = input.slowestWords.join(', ') || 'none'
  return [
    `Generate a ${input.mode === 'quotes' ? 'quote-style' : 'general'} practice passage of about 60 words.`,
    `Focus letters to practise: ${letters}.`,
    `Words the user was slow on: ${words}.`,
    `The user's current speed is about ${input.netWpm} WPM at ${input.accuracy}% accuracy.`,
  ].join(' ')
}
```

```ts
import type { CoachInputShape } from './prompts'

const NUMERIC_FIELDS: (keyof CoachInputShape)[] = [
  'netWpm', 'grossWpm', 'accuracy', 'consistency', 'backspaceCount', 'recentAverageWpm',
]

export function validateCoachInput(
  body: unknown,
): { ok: true; value: CoachInputShape } | { ok: false; error: string } {
  if (typeof body !== 'object' || body === null) {
    return { ok: false, error: 'Body must be a JSON object.' }
  }
  const b = body as Record<string, unknown>

  for (const f of NUMERIC_FIELDS) {
    if (typeof b[f] !== 'number' || Number.isNaN(b[f])) {
      return { ok: false, error: `Field "${f}" must be a number.` }
    }
  }
  if (!Array.isArray(b.weakLetters) || !b.weakLetters.every((x) => typeof x === 'string')) {
    return { ok: false, error: 'Field "weakLetters" must be an array of strings.' }
  }
  if (!Array.isArray(b.slowestWords) || !b.slowestWords.every((x) => typeof x === 'string')) {
    return { ok: false, error: 'Field "slowestWords" must be an array of strings.' }
  }
  if (b.mode !== 'words' && b.mode !== 'quotes') {
    return { ok: false, error: 'Field "mode" must be "words" or "quotes".' }
  }
  if (b.durationSeconds !== 30 && b.durationSeconds !== 60) {
    return { ok: false, error: 'Field "durationSeconds" must be 30 or 60.' }
  }
  return { ok: true, value: b as unknown as CoachInputShape }
}
```

> **Verify before trusting this shape.** `@google/genai` evolves; confirm the method and field names against the installed package's types (`node_modules/@google/genai`) or current docs (https://ai.google.dev/gemini-api/docs) — as of this plan the Interactions API is `ai.interactions.create({ model, system_instruction, input, response_format })` returning `.output_text`.
```ts
import { GoogleGenAI } from '@google/genai'

const MODEL = process.env.COACH_MODEL ?? 'gemini-3.6-flash'

export class NotConfiguredError extends Error {
  constructor() {
    super('GEMINI_API_KEY is not set on the server.')
    this.name = 'NotConfiguredError'
  }
}

let client: GoogleGenAI | null = null
function getClient(): GoogleGenAI {
  const apiKey = process.env.GEMINI_API_KEY
  if (!apiKey) throw new NotConfiguredError()
  if (!client) client = new GoogleGenAI({ apiKey })
  return client
}

export async function generateStructured(system: string, input: string, schema: object): Promise<string> {
  const res = await getClient().interactions.create({
    model: MODEL,
    system_instruction: system,
    input,
    response_format: { type: 'text', mime_type: 'application/json', schema },
  })
  return res.output_text
}

export async function generateText(system: string, input: string): Promise<string> {
  const res = await getClient().interactions.create({
    model: MODEL,
    system_instruction: system,
    input,
  })
  return res.output_text
}
```

```ts
import type { Response } from 'express'
import { NotConfiguredError } from './gemini'

export function sendAiError(res: Response, e: unknown): void {
  if (e instanceof NotConfiguredError) {
    res.status(503).json({ error: 'AI is not configured on the server.', code: 'not_configured' })
    return
  }
  res.status(502).json({ error: 'The AI request failed. Please try again.', code: 'ai_error' })
}
```

```ts
import { validateCoachInput } from '../validate'
import { ANALYSIS_SYSTEM, buildAnalysisInput } from '../prompts'

import { generateStructured } from '../gemini'

export const analyzeRouter = Router()

analyzeRouter.post('/api/analyze', async (req, res) => {

  if (!parsed.ok) {

    sendAiError(res, e)
  }
})
```

```ts
import type { Metrics } from '../engine/metrics'
import type { Mode, Duration } from '../engine/types'
import type { TypingSession } from '../storage/history'

export type CoachInput = {
  netWpm: number
  grossWpm: number
  accuracy: number
  consistency: number
  weakLetters: string[]
  slowestWords: string[]
  backspaceCount: number
  recentAverageWpm: number
  mode: Mode
  durationSeconds: Duration
}

export function buildCoachInput(
  metrics: Metrics,
  mode: Mode,
  durationSeconds: Duration,
  history: TypingSession[],
): CoachInput {
  const weakLetters = Object.entries(metrics.mistypedLetters)
    .sort((a, b) => b[1] - a[1])
    .slice(0, 5)
    .map(([letter]) => letter)

  const recentAverageWpm = history.length
    ? Math.round(history.reduce((sum, s) => sum + s.netWpm, 0) / history.length)
    : 0

  return {
    netWpm: metrics.netWpm,
    grossWpm: metrics.grossWpm,
    accuracy: metrics.accuracy,
    consistency: metrics.consistency,
    weakLetters,
    slowestWords: metrics.slowestWords.map((w) => w.word),
    backspaceCount: metrics.backspaceCount,
    recentAverageWpm,
    mode,
    durationSeconds,
  }
}
```

```ts
import type { CoachInput } from './summary'

export type CoachAnalysis = {
  strength: string
  weakness: string
  recommendation: string
  exerciseType: string
}

const TIMEOUT_MS = 15_000

async function postJson<T>(path: string, input: CoachInput): Promise<T> {
  const controller = new AbortController()
  const timer = setTimeout(() => controller.abort(), TIMEOUT_MS)
  try {
    const res = await fetch(path, {
      method: 'POST',
      headers: { 'content-type': 'application/json' },
      body: JSON.stringify(input),
      signal: controller.signal,
    })
    if (!res.ok) {
      const detail = await res.json().catch(() => null)
      throw new Error(detail?.error ?? `Request to ${path} failed (${res.status}).`)
    }
    return (await res.json()) as T
  } finally {
    clearTimeout(timer)
  }
}

export function analyze(input: CoachInput): Promise<CoachAnalysis> {
  return postJson<CoachAnalysis>('/api/analyze', input)
}

export function generatePassage(input: CoachInput): Promise<{ passage: string; focus: string[] }> {
  return postJson<{ passage: string; focus: string[] }>('/api/passage', input)
}
```

```tsx
import { useEffect, useState } from 'react'
import { analyze, type CoachAnalysis } from '../../api/coach'
import type { CoachInput } from '../../api/summary'

type Props = {
  coachInput: CoachInput
  onStartExercise?: () => void
}

type State =
  | { status: 'loading' }
  | { status: 'ready'; analysis: CoachAnalysis }
  | { status: 'error' }

export function AiCoachCard({ coachInput, onStartExercise }: Props) {
  const [state, setState] = useState<State>({ status: 'loading' })

  useEffect(() => {
    let cancelled = false
    setState({ status: 'loading' })
    analyze(coachInput)
      .then((analysis) => { if (!cancelled) setState({ status: 'ready', analysis }) })
      .catch(() => { if (!cancelled) setState({ status: 'error' }) })
    return () => { cancelled = true }
  }, [coachInput])

  return (
    <div className="rounded-lg bg-surface p-4">
      <h3 className="mb-2 text-sm uppercase tracking-wide text-accent-soft">AI coach</h3>

      {state.status === 'loading' && (
        <div data-testid="coach-loading" className="space-y-2 animate-pulse">
          <div className="h-3 w-2/3 rounded bg-muted/30" />
          <div className="h-3 w-1/2 rounded bg-muted/30" />
        </div>
      )}

      {state.status === 'error' && (
        <p className="text-muted">Coach is unavailable right now — your results are saved.</p>
      )}

      {state.status === 'ready' && (
        <div className="space-y-2">
          <p><span className="text-correct">Strength: </span>{state.analysis.strength}</p>
          <p><span className="text-error">Weakness: </span>{state.analysis.weakness}</p>
          <p className="text-muted">{state.analysis.recommendation}</p>
          {onStartExercise && (
            <button type="button" onClick={onStartExercise}
              className="mt-2 rounded-lg bg-accent px-4 py-2 text-white hover:bg-accent-soft">
              Start recommended exercise
            </button>
          )}
        </div>
      )}
    </div>
  )
}
```
